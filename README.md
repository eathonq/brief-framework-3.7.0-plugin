## 项目目录结构
```
├── assets              // 库资源文件夹
|   ├── base            // ts基础代码
│   ├── common          // 常用基础组件
|   ├── expand          // 扩展组件或代码（未分类）
│   ├── guide           // 引导管理
│   ├── i18n            // 多语言管理
│   ├── mvvm            // MVVM框架
│   ├── platform        // 平台代码管理
│   ├── ui_test         // UI测试框架
|   ├── Brief.ts        // brief库入口
```
